# Simulacro Nacional 2026 - Así funcionará la nueva alerta de emergencia en celulares

La **Coordinación Nacional de Protección Civil (CNPC)** activará por primera vez un sistema de alerta masiva directamente en teléfonos móviles durante el **Primer Simulacro Nacional 2026**, el cual se llevará a cabo el próximo **6 de mayo**.

## Tecnología Cell Broadcast - Alertamiento sin saturación de redes

El ejercicio pondrá a prueba la tecnología **Cell Broadcast**, un sistema de transmisión que permite el envío de mensajes a todos los dispositivos dentro de un área geográfica específica. A diferencia de los mensajes de texto convencionales, este método **no satura las líneas telefónicas** y garantiza una recepción inmediata.

Según las especificaciones de la **GSMA**, las características técnicas de esta herramienta son determinantes para la seguridad pública:
*   **No requiere conexión a internet** ni el uso de datos móviles para funcionar.
*   Llega a todos los equipos activos en la zona de cobertura de forma simultánea.
*   El mensaje se despliega incluso si el teléfono se encuentra en **modo silencio o bloqueado**.

## Configuración y pasos para garantizar la recepción

Para que el dispositivo sea capaz de procesar este aviso de emergencia, es fundamental que los usuarios verifiquen la configuración de sus terminales. La **CNPC** recomienda realizar este ajuste preventivo antes del **6 de mayo** para asegurar la efectividad del ejercicio.

Los usuarios deben ingresar a la sección de **Ajustes** o **Configuración** de su teléfono, localizar el apartado de **Seguridad y emergencia** (o **Notificaciones**) y comprobar que las **Alertas de emergencia inalámbricas** o **Alertas gubernamentales** se encuentren activas.
