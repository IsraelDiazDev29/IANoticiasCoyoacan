# Telcel AT&T Movistar y Bait reasignarán líneas inactivas a nuevos usuarios

Las operadoras **Telcel**, **AT&T**, **Movistar** y el operador virtual **Bait** tienen la facultad de cancelar líneas y reasignar números telefónicos a nuevos clientes en **México**. Esta medida se aplica principalmente a cuentas de prepago que no registran actividad económica o uso durante periodos prolongados.

## El proceso de cancelación por inactividad

Las empresas de telecomunicaciones interpretan la falta de **recargas de saldo**, incluso por montos mínimos, como una señal de abandono del servicio por parte del titular. Para optimizar el inventario numérico coordinado por las autoridades mexicanas, las compañías siguen un protocolo gradual de recuperación de la línea.

*   **Notificación previa**: Las empresas envían avisos de inactividad al usuario antes de proceder con cualquier restricción.
*   **Suspensión temporal**: Se inhabilita la capacidad de realizar llamadas o usar datos, manteniendo el número reservado por un tiempo breve.
*   **Cancelación definitiva**: El contrato se da por terminado y el número queda disponible para que la autoridad lo asigne a un nuevo contrato.

## Impacto en la conectividad del usuario

Perder una línea telefónica no solo implica el cambio de número, sino el riesgo de perder acceso a servicios vinculados como **banca móvil**, redes sociales y sistemas de verificación de dos pasos. Aunque en **México** existe el derecho a la portabilidad numérica, este solo es válido mientras la línea se mantenga activa y con el pago correspondiente.
