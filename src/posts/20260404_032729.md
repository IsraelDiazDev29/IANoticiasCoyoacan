# Microsoft elimina aplicaciones web en Windows 11 para priorizar el rendimiento nativo

**Microsoft** confirmó un cambio de estrategia radical para **Windows 11** al anunciar la creación de un equipo dedicado al desarrollo de **aplicaciones 100% nativas**. La medida busca revertir años de dependencia en tecnologías web que han degradado la experiencia del usuario y el rendimiento del sistema.

## El fin de las aplicaciones híbridas y lentas

La decisión surge tras múltiples quejas sobre la capacidad de respuesta y el **uso excesivo de memoria RAM** en herramientas integradas. Durante los últimos años, Microsoft impulsó el uso de aplicaciones web progresivas (PWA), un movimiento que ahora se considera un obstáculo para la fluidez del entorno.

**Rudy Huyn**, arquitecto asociado de Microsoft, liderará este nuevo equipo enfocado en experiencias nativas. El objetivo es que programas básicos dejen de funcionar bajo envoltorios de navegadores web y recuperen la integración total con la arquitectura de **Windows**.

* El cambio afectará a herramientas clave como el **Explorador de archivos** y el **Menú Inicio**.
* Se priorizará la tecnología **WinUI** para garantizar una interfaz coherente y rápida.
* Aplicaciones criticadas por su lentitud, como las basadas en **Chromium**, serán evaluadas para su reconstrucción.

![Logotipo de Microsoft y entorno de desarrollo corporativo](/assets/img/microsoft-corporativo-x7v3b9n2.webp)

## Impacto en la productividad y el sistema central

Esta reestructuración coincide con una actualización mayor de **Windows 11** diseñada para reducir los tiempos de carga. La empresa admitió implícitamente que la transición hacia lo web no cumplió con las expectativas de rendimiento que los usuarios demandaban para un sistema operativo moderno.

El rediseño también incluirá mejoras en la **barra de tareas** y una optimización en los menús contextuales. Con este giro, la administración actual de Windows, encabezada por **Pavan Davuluri**, busca cumplir la promesa de hacer que los usuarios prefieran el sistema por su calidad técnica y no solo por necesidad.
