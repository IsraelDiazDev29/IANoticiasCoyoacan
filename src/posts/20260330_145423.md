# Google Drive integra escáner inteligente con IA para procesar documentos en segundos

**Google Drive** ha implementado una nueva función de **inteligencia artificial** que permite escanear y digitalizar documentos físicos de forma automatizada. Esta herramienta, disponible para dispositivos **Android**, optimiza tareas de oficina que antes requerían software especializado o procesos manuales prolongados.

## Digitalización de alta precisión y modo continuo

A diferencia de las versiones anteriores, la actualización utiliza algoritmos avanzados para detectar **bordes de página con precisión milimétrica**. El sistema permite realizar **capturas multipágina en tiempo real**, lo que facilita el procesamiento de grandes volúmenes de papel sin interrupciones.

*   **Modo continuo:** La IA identifica cada página de una pila de documentos y las enumera automáticamente.
*   **Formato de salida:** El sistema genera archivos en **PDF o JPG** listos para ser compartidos en la nube.
*   **Accesibilidad:** No se requiere una suscripción a **Google AI Pro** para utilizar esta mejora técnica.

![Proceso de escaneo asistido por inteligencia artificial en dispositivos móviles](/assets/img/image004-z3v5b7n9.webp)

## Cómo activar la función en dispositivos Android

Para utilizar esta herramienta, el usuario debe abrir la aplicación de **Drive** y seleccionar el icono **"+"** ubicado en la parte inferior. Al elegir la opción **Escanear**, la cámara activa el reconocimiento automático de texto y formas, permitiendo un ajuste inmediato de la legibilidad mediante filtros.

Esta integración de **Gemini** y otras tecnologías de Google en su ecosistema de almacenamiento busca reducir el tiempo de gestión administrativa. La función incluye herramientas de **edición rápida** para recortar elementos no deseados o mejorar el contraste del documento digitalizado antes de su guardado final.
